# Elin Mod

[Elin](https://store.steampowered.com/app/2135150/Elin/)向けに作成したModです。
Steam Workshopで公開していないModも含まれます。

## ビルド
ビルドの実行前に、Elinのインストール先フォルダのパスを設定した環境変数`ElinGamePath`を設定してください。

また前提としてElin Modの開発環境が必要です。環境構築については[Elin Modding Wiki](https://elin-modding-resources.github.io/Elin.Docs/)を参考にしてください。
